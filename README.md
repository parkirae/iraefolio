## 📝 포트폴리오 소개

**정보 공유**

두 사람 이상이 하나의 정보를 공동으로 소유하는 것 입니다. 정보 공유는 다양한 이점을 가집니다. 첫째, 지식 확장입니다. 둘째, 문제 해결입니다. 셋째, 창의성 촉진입니다. 이러한 정보 공유의 이점이 월드 와이드 웹을 탄생시켰습니다.

<br>
iraefolio는 정보 공유를 위한 도구입니다. iraefolio를 사용하는 동안 사용자는 지식이 확장되고, 문제가 해결되고, 창의성이 촉진될 것입니다. 구체적으로 사용자는 공유 정보를 열람할 수 있고, 자신의 정보를 공유할 수도 있습니다. 또한 간편하고 직관적인 인터페이스를 제공함으로써 누구나 쉽게 사용할 수 있습니다. iraefolio는 단순하지만 강력한 도구입니다. 

<br>
<br>

## 🌁 프로젝트 배경

3년간, 신문사에 근무하며 수많은 웹 문서를 작성했습니다. 회사의 CMS를 이용했는데 매번 불편함을 느꼈습니다. iraefolio에서는 웹 문서를 작성하면서 발생한 불편함을 최소화하고자 했습니다. 구체적 방안은 다음과 같습니다.

**첫째, 비동기통신입니다.**
웹 페이지의 변화 없이 CRUD를 구현했습니다.

**둘째, 드래그앤드롭(DND)입니다.**
단순히 파일을 드래그함으로써 웹 문서에 첨부되게 했습니다.

**셋째, 직관적인 인터페이스입니다.**
누구나 쉽게 이해하고 사용할 수 있게 했습니다.

## 🔧 Stack

**Frontend(Web)**
- **Language** : JavaScript
- **Library** : jQuery, moment, toast UI grid, toast UI editor, toast UI pagination
<br />

**Backend**
- **Language** : Java 
- **Framework** : Spring Boot 2.7.5, Spring Security 2.7.5
- **Database** : MariaDB
- **ORM** : myBatis
- **BUILD** : gradle
- **Deploy**: Jenkins, github webhook
